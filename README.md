# Networking Fundamentals Labs
**COP2830 — Cisco Networking Fundamentals | Santa Fe College**

Hands-on networking labs using Cisco Packet Tracer. Covers IP addressing, subnetting, routing, switching, and troubleshooting.

---

## Topics Covered

| Topic | Description |
|-------|-------------|
| IP Addressing | IPv4 addressing, classes, and assignment |
| Subnetting | CIDR notation, subnet masks, host calculation |
| OSI Model | Layer-by-layer breakdown and practical mapping |
| Routing | Static routing, default routes, routing tables |
| Switching | VLANs, trunk ports, MAC address tables |
| Troubleshooting | Ping, traceroute, connectivity diagnosis |

---

## Tools Used

- **Cisco Packet Tracer** — Network simulation
- **Windows / Linux CLI** — Command line tools
- `ping` `tracert` `ipconfig` `ifconfig` `nslookup`

---

## Key Concepts

```
IP Address:   192.168.1.0 / 24
Subnet Mask:  255.255.255.0
Hosts:        254 usable
Broadcast:    192.168.1.255
```
